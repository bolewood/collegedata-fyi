# Takedown Log

This file records every takedown request the archive has received and the
outcome of each. It is maintained as a public transparency measure per
[ADR 0008](decisions/0008-takedown-process.md).

## Format

Each row: `YYYY-MM-DD | school_id | cds_year | reason_category | outcome`

- **reason_category**: `ferpa_adjacent`, `accidental_publish`,
  `version_dispute`, `attribution_dispute`, or `other`.
- **outcome**: `catalog_removed`, `catalog_and_bytes_removed`,
  `declined`, or `restored`.

No requester PII (names, emails, correspondence) is recorded.
