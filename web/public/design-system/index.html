collegedata.fyi · Design system · v0.1

Paper, ink, and one quiet green.

A warm archival system for an open Common Data Set archive. IBM Plex Serif display, IBM Plex Sans body, muted forest accent, and a library of small components built for reading data — not selling software.

Radius & shadow

§ Usage

Principles

  1. Read like a library, not a SaaS landing page.
  2. One accent, used rarely. When forest appears, it means this.
  3. Numbers are always tabular, always next to their source section (§C, §H…).
  4. Rules, not shadows. Density earned, not decorated.
  5. Never hide a source. Every number links back to its CDS cell.
